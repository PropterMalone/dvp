# Session Wrap

1. Save any new information you learned about the user's preferences, interests, or ongoing work to memory files in `~/.claude/`
2. Note anything that was left unfinished or that should be picked up next time
3. Brief summary of what was accomplished

Keep it concise. Don't over-document.
